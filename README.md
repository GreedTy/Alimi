# Alimi 스터디그룹관리앱
Springboot 2.5.x
Security, JPA-querydsl, lombok, webjar(JQuery, Bootstrap), Modelmapper

CI/CD : Docker-Gitlab(Gitlab-Runner,docker-compose), docker registry, springboot-jar 배포
